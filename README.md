# TATA Forage Data Visualisation Project

## Overview
Completed the TATA Insights and Quants (iQ) Data Visualisation Job Simulation on Forage.

This project focuses on analyzing an online retail dataset using Power BI to generate strategic business insights for senior leadership.

## Tools Used
- Power BI
- Excel / CSV
- Data Cleaning
- Business Analytics
- Dashboard Design

## Key Insights
- Q4 generated highest revenue, especially November
- EIRE, Netherlands, Germany are top markets
- Revenue heavily dependent on top customers
- Seasonal demand patterns identified
- Global expansion opportunities highlighted

## Dashboard Visuals
- Revenue Time Series
- Top Countries by Revenue
- Top Customers Analysis
- Global Demand Map

## Files Included
- Power BI Dashboard (.pbix)
- Final Report (.pdf)
- Dataset
- Dashboard Screenshots

## Business Recommendations
- Increase inventory before Q4
- Expand in Europe markets
- Improve VIP customer retention
- Strengthen low-performing markets
